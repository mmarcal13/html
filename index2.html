Div e Span

A tag `<div>` é um container genérico de bloco, enquanto `<span>` é um container genérico de linha. Ambos são muito usados com CSS.

Este é um conteúdo dentro de um div, com um span embutido.

O `<span>` é ótimo para aplicar estilos a partes específicas de um texto.
